# ProyectoProgramacionAvanzada-
Proyecto para la clase de programacion avanzada con el tema "Sistema de Gestión de Actividades en Clubes Deportivos"" 


Gestión de Socios: Registro, edición, búsqueda y eliminación de socios con estados y categorías (Básico, Platino, Diamante).

Gestión de Clubes y Actividades: Estructura anidada donde cada club posee una colección de actividades deportivas.

Sistema de Reservas: Funcionalidad de negocio que vincula socios con actividades, validando cupos y pertenencia al club.Persistencia de Datos: Carga y guardado automático mediante archivos de texto (CSV/TXT).

Interfaz Dual: Soporte para interacción vía consola y ventanas gráficas (Swing)

Lenguaje: Java (Oracle JDK 11).


IDE: Netbeans 21 / Eclipse.


Control de Versiones: Git & GitHub
